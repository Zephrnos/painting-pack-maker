# Contributing

Thanks for contributing! Please follow these steps:

1. Fork the repository and create a branch:
   `git checkout -b feat/short-description` or `git checkout -b fix/short-description`
2. Make small, focused changes with clear commit messages.
3. Run the app locally and test your changes.
4. Open a Pull Request targeting `main` with a clear description and testing steps.

## Code Style
- Keep commits atomic.
- Write tests when possible.
