# MCPPM
Rust and Tauri based painting pack maker for Minecraft uses native webview within Tauri.

![](example.png)

# To Use:

 - Click "File" And import images
 - Select and deselect photos you want in your painting pack
 - Give paintings names and artists, and fill out other fields
 - Select Pack -> Export

 And you've just made your very own painting pack for Minecraft!
